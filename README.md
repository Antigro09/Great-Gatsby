# The Great Gatsby - Interactive Literary Analysis

A Next.js + Tailwind CSS website focused on close reading of The Great Gatsby by F. Scott Fitzgerald.

## Sections Included

- Home
- Characters
- Symbols
- Themes
- Important Scenes
- Quotes
- Final Reflection

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Build Check

```bash
npm run build
```

The project is configured as a static-rendered app for all analysis routes.
